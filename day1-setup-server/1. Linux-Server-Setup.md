
1. Tujuan
- Menyiapkan server dari 0
- Membuat user baru dan mengaktifkan SSH
- Melatih dasar administrasi sistem


2. Environment/Lingkungan
- VM : QEMU
- OS : Ubuntu Server
- Resource : 2 Core, 1 GB RAM
- Tools : SSH, systemctl, journalctl, htop

3. Langkah Kerja
- Mengganti hostname
```bash
sudo hostnamectl set-hostname homelab
```

- Membuat user baru
```bash
sudo adduser newuser
sudo passwd newuser
sudo usermod -aG sudo newuser
```

- Menginstall htop
```bash
sudo apt update && sudo apt upgrade
sudo apt install htop
```

- Cek status dan menyalakan ssh apabila belum start/disable
```bash
sudo systemctl status ssh

sudo systemctl enable/start ssh
```

- Cek ip address homelab/ubuntu server(guest os)
```bash
ip a
```

- Testing ssh dari terminal dari host 
```bash
ssh user@192.168.123.68
```
